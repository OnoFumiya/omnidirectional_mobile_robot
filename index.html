独立4輪ステアリング機構を用いた

全方向移動ロボットの車輪制御

小野郁哉
創価大学
創価大学卒業研究（2023年度）
卒業論文 Code X(議論)

概要

著者が所属する大学の研究室（創価大学 崔研究室）では，全方向移動ロボットにSOBIT PROを作成した． このロボットは独立４輪ステアリング機構を用いている． この独立４輪ステアリングロボットでは，把持の際に物体に対して姿勢を保ったまま移動することができたり，差動二輪機構のロボットでは実現できなかった動きの再現が可能となった． しかし，自律移動といった細かい移動に対しては，これまで差動二輪機構と同じ動きしかしてこなかった． そこで今回，独立４輪機構を自律移動にも活かすための4輪の独立した制御とそれを用いた研究開発を行う． 具体的には差動2輪モデルでの従来の自律移動に加え，歩行者といった逐次回避の必要な障害物の回避に横方向にDynamic Window Approachを適応し，一時的な移動方向の拡張をする制御を行う．

本研究の社会的背景

ロボットが自律的に人の生活を支援することが求められている． 人にものを運んだり，人の荷物運搬を代替するなど幅広い活躍が期待できる． それらの生活支援には，ロボットの柔軟な移動が不可欠である． 例えば物体を把持する場合に，ロボット全体が物体の把持姿勢に合わせることは，マニピュレーションに追加の自由度を持たせることができる． また，自律移動における経路変更に対して，差動二輪機構のようなその場回転のみの対応では，移動に遅れてしまうという課題がある． 特に，移動するワゴンや歩行者など逐次回避する必要のある障害物がある環境でロボットが移動をする場合，経路変更に対して，迅速かつ柔軟に回避する必要がある． そこで本文では，SOBIT PROの制御を中心とした新たな車輪制御の手法と，その制御を用いた歩行者回避の検証実験について述べる．

従来の課題

これまで，SOBIT PROは自律移動において，差動二輪と同様の速度制御を行っていた． ロボットへ速度が送られてきた際に，４輪を上の図に示すように制御していた． 特に，カーブをする速度の場合，自動車のように前方の車輪の方向を切り替えることで実現した． しかし，この制御にはいくつか課題がある． この自動車のような車輪制御では，進行方向の円と車輪で若干の誤差が生じてしまい，車輪に無駄な力が発生する． ロボット自体の進行方向と，それぞれの車輪の円が同心円とならないことが原因としてあげられる． これにより，車輪の破損や車輪からのフィードバック(Odometry)の誤差が大きくなってしまった． また，この制御では直線移動を軸としたカーブにしか対応できておらず，全方向移動ロボットの利点を活かしきれていないことがあげられる． つまり，独立４輪ステアリング機構を用いて，差動二輪の動きを再現している状況であることがわかる． そこで，独立４輪ステアリング機構を用いた全方向移動による自律移動をするための制御を目的とする．

ロボットの回転から車輪を接線方向にする方向制御と速度制御

差動二輪モデルでの制御をしている課題を解決するために，４つの車輪それぞれが向くべき方向を算出する必要がある． 本手法の焦点はカーブにおける全車輪の方向および速度であるため，カーブの速度が送られてきたことを前提とする． まずロボットに送られてくる速度からロボット全体の旋回についてを，速度の大きさと中心角の角速度という面で算出する． 具体的には，ROS/ROS2における速度型メッセージ(geometry_msgs/msg/Twist型)のうち，linear.xとlinear.y，angular.zについての3要素から求まる． こうして得られた速度と中心角速度から，ロボットが動くべき円の中心座標を算出する． この中心座標をすべての車輪から見て同心円上の接線方向に向くことで，ロボットの動きに合わせた車輪の制御となる． これにより，車輪への方向の誤差による負荷が少なくなる． さらに，円の中心座標とロボットの距離，円の中心座標と各車輪との距離，から比を速度の大きさに掛け合わせることで，各車輪ごとに速度を決定することができる． これにより，車輪の速さが各車輪に求められている大きさとなり，滑りによるOdometryの誤差を最小にすることができる． これらのアルゴリズムを下の動画で示す．

歩行者回避の実験

この手法を用いて，歩行者を回避する実験を行った． 歩行者はロボットの前方から歩行してくるものとして，ロボットは道を譲らなくてはならない状況を想定する． SOBIT PROには，センサーが前方に搭載されている． そのため，今回用いる自律移動では前方の障害物が検知できるようにするため，基本的に差動二輪での動作をする． しかし，歩行者を回避するときは横方向の速度を一時的に出力する移動手法を用いる． これは，「基本的にセンサーに有利な移動をするが，歩行者を回避する場合は回避の迅速さを優先」することに重きを置いている． この実験の様子を下の動画に示す．
